High-Speed Rail Traction Power Analysis Software

Designed and developed a custom Simulation Software System for analyzing traction power supply behavior for Mumbai Ahmedabad Bullet Train. This work was carried out as part of a sponsored research and development initiative at the Industrial Research and Consultancy Centre (IRCC), IIT Bombay in collaboration with High Speed Railway India Corporation (HSRCL).

The solution was implemented as an integrated platform combining a Qt-based C++ application with Octave-driven numerical computation, enabling comprehensive electrical analysis of railway traction systems.

  • Architected and implemented both front-end and back-end components using C++ and Qt.
  • Conducted R&D for electrical logic tailored to high-speed rail traction power requirements.
  • Integrated Octave-based computational models into the C++ application.
  • Implemented load flow, short-circuit, and harmonics analysis.
  • Consumed railway timetables and route-level parameters for scenario evaluation.
  • Developed and validated the software in a Linux-based environment.

Project Sponsor: High Speed Railways India Corporation (HSRCL)

Technologies & Domains: C++, Qt, Octave, Linux, traction power systems, electrical power system analysis

COMTRADE-Based Data Acquisition & Visualization Tool (iCoReader)

Designed and developed a data acquisition and visualization tool for Distributed Disturbance Recorder (DDR) systems used in power system monitoring and analysis.

  • Implemented a COMTRADE file reader compliant with IEEE standards.
  • Developed a Qt-based C++ desktop application for parsing and visualization.
  • Enabled detailed post-event disturbance analysis.
  • Integrated as a module within the existing iPDC suite.
  • Improved usability over conventional COMTRADE viewers.

Standard: IEEE C37.118.1-2013 (COMTRADE)

Technologies & Domains: C++, Qt, COMTRADE, power system monitoring, data acquisition and visualization, Linux

PMU Connection Tester & Synchrophasor Data Visualization Tool (iTester)

Developed a Linux-based desktop application to test, validate, and visualize Phasor Measurement Unit (PMU) data using synchrophasor technology.

  • Developed a PMU connection testing tool using C++ and Qt.
  • Implemented real-time acquisition of PMU data streams.
  • Parsed IEEE synchrophasor protocol packets.
  • Displayed raw hex packets and decoded values.
  • Rendered real-time plots for frequency, magnitude, angle, and analogs.
  • Designed an intuitive multi-plot UI.

Standard: IEEE C37.118.2-2011 (Synchrophasor Data Transfer)

Technologies & Domains: C++, Qt, Linux, synchrophasor technology, real-time data acquisition, protocol parsing
