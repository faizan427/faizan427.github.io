High-Speed Rail Traction Power Analysis Software

Designed and developed a custom software application for analyzing traction power supply systems for high-speed rail corridors. The project was carried out as a sponsored research and development initiative at the Industrial Research and Consultancy Centre (IRCC), IIT Bombay.

The software was developed as an integrated solution combining a Qt-based C++ application with Octave-based computational models, enabling advanced electrical analysis of railway traction systems.

  • Designed and implemented the front-end and back-end of the application using C++ with the Qt framework.
  • Conducted research and development (R&D) activities for electrical logic design specific to high-speed rail traction power systems.
  • Integrated Octave-based numerical models with the Qt C++ application to create a unified analysis workflow.
  • Implemented electrical analysis capabilities including load flow analysis, short-circuit analysis, and harmonics analysis.
  • Designed the software to accept railway timetable data and route-level parameters as inputs for scenario-based analysis.
  • Worked in a Linux-based development environment following structured research, validation, and documentation practices.

Project Sponsor: High Speed Railways India Corporation (HSRCL)

Technologies & Domains: C++, Qt, Octave, Linux, traction power systems, electrical power system analysis
