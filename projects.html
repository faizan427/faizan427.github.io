High-Speed Rail Traction Power Analysis Software

Designed and developed a custom software application for analyzing traction power supply systems for high-speed rail corridors. The project was carried out as a sponsored research and development initiative at the Industrial Research and Consultancy Centre (IRCC), IIT Bombay.

The software was developed as an integrated solution combining a Qt-based C++ application with Octave-based computational models, enabling advanced electrical analysis of railway traction systems.

  • Designed and implemented the front-end and back-end using C++ and Qt.
  • Performed R&D for electrical logic specific to traction power systems.
  • Integrated Octave-based numerical models with the Qt C++ application.
  • Implemented load flow, short-circuit, and harmonics analysis.
  • Processed railway timetable data and route parameters as inputs.
  • Worked in a Linux-based environment with structured validation practices.

Project Sponsor: High Speed Railways India Corporation (HSRCL)

Technologies & Domains: C++, Qt, Octave, Linux, traction power systems, electrical power system analysis

COMTRADE-Based Data Acquisition & Visualization Tool (iCoReader)

Developed a data acquisition and visualization tool for Distributed Disturbance Recorder (DDR) systems used in power system monitoring applications.

  • Designed and implemented a COMTRADE file reader compliant with IEEE standards.
  • Developed a desktop application named iCoReader using C++ and Qt.
  • Parsed, processed, and visualized power system disturbance data.
  • Integrated iCoReader as a new module within the existing iPDC suite.
  • Enhanced data visualization and analysis compared to conventional COMTRADE viewers.

Standards: IEEE C37.118.1-2013 (COMTRADE)

Technologies & Domains: C++, Qt, COMTRADE, power system monitoring, data acquisition and visualization, Linux

PMU Connection Tester & Synchrophasor Data Visualization Tool (iTester)

Developed a Linux-based desktop application to test and validate Phasor Measurement Unit (PMU) connections using Synchrophasor technology.

  • Developed iTester, a PMU connection testing tool using C++ and Qt.
  • Implemented live acquisition of raw PMU data from simulators.
  • Parsed binary data streams according to IEEE synchrophasor standards.
  • Displayed raw hex packets and decoded values in human-readable form.
  • Implemented real-time plots of frequency, magnitude, angle, and analog values.
  • Designed a clear UI with multiple plotting areas and real-time indicators.

Standards: IEEE C37.118.2-2011 (Synchrophasor Data Transfer)

Technologies & Domains: C++, Qt, Linux, synchrophasor technology, real-time data acquisition, protocol parsing
