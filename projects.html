High-Speed Rail Traction Power Analysis Software

Designed and developed a custom software system for analyzing traction power supply behavior in high-speed rail corridors. This work was carried out as part of a sponsored research and development initiative at the Industrial Research and Consultancy Centre (IRCC), IIT Bombay.

The solution was implemented as an integrated platform combining a Qt-based C++ application with Octave-driven numerical computation, enabling comprehensive electrical analysis of railway traction systems.

  • Architected and implemented both front-end and back-end components using C++ and the Qt framework.
  • Conducted research and development for electrical logic tailored to high-speed rail traction power requirements.
  • Integrated Octave-based computational models into the C++ application to support complex numerical analysis workflows.
  • Implemented core electrical studies including load flow analysis, short-circuit analysis, and harmonics analysis.
  • Designed the system to consume railway timetables and route-level parameters as inputs for scenario-based evaluation.
  • Developed and validated the software in a Linux-based environment following structured testing and documentation practices.

Project Sponsor: High Speed Railways India Corporation (HSRCL)

Technologies & Domains: C++, Qt, Octave, Linux, traction power systems, electrical power system analysis

COMTRADE-Based Data Acquisition & Visualization Tool (iCoReader)

Designed and developed a data acquisition and visualization tool for Distributed Disturbance Recorder (DDR) systems used in power system monitoring and analysis.

  • Implemented a COMTRADE file reader compliant with IEEE standards for power system disturbance data.
  • Developed a desktop application named iCoReader using C++ and Qt for data parsing and visualization.
  • Parsed, processed, and visualized disturbance records to enable detailed post-event analysis.
  • Integrated iCoReader as a functional module within the existing iPDC suite.
  • Improved usability and analytical capability compared to conventional COMTRADE viewers.

Standard: IEEE C37.118.1-2013 (COMTRADE)

Technologies & Domains: C++, Qt, COMTRADE, power system monitoring, data acquisition and visualization, Linux

PMU Connection Tester & Synchrophasor Data Visualization Tool (iTester)

Developed a Linux-based desktop application to test, validate, and visualize Phasor Measurement Unit (PMU) data using synchrophasor technology.

  • Developed iTester, a PMU connection testing application using C++ and Qt.
  • Implemented real-time acquisition of raw PMU data streams from simulators.
  • Parsed binary data packets according to IEEE synchrophasor communication standards.
  • Displayed raw hexadecimal packets alongside decoded, human-readable values.
  • Implemented real-time plots for frequency, magnitude, phase angle, and analog parameters.
  • Designed a clear and intuitive UI with multiple plotting regions and real-time indicators.

Standard: IEEE C37.118.2-2011 (Synchrophasor Data Transfer)

Technologies & Domains: C++, Qt, Linux, synchrophasor technology, real-time data acquisition, protocol parsing
